F5 — Instructor-Led Training

A 3-day SaaS training course, built to stay current

Material created for the 3-day training course for SaaS product "F5 Distributed Cloud Services" was designed and created by me in an authoring tool called Xyleme (aka "Madcap Xyleme"). The course is designed to be taught by an instructor via Zoom or in-person, and contains reference material and labs. Due to the ever-changing nature of the product itself, the material is designed to be constantly live and constantly updated, so students always have the most-recent version on-hand.

Each link below is a separate web page for three different purposes:

  • Instructor Guide — shows everything the instructor needs to know to run the class, including secret tips and tricks for using SaaS tools and AWS provisioning.
  • Presentation — displays in Xyleme's version of PowerPoint, and is meant to be presented to the class during training.
  • Student Guide — nearly identical to the Instructor Guide except for excised sensitive content (e.g. AWS access keys).

Note: some of the images in these files do not render properly due to internal reference issues. This is expected but has no bearing on content.
